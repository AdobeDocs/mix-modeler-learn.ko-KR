# 모델 드리프트 감지

Mix Modeler의 모델 드리프트 감지 기능을 사용하면 모델의 드리프트가 자동으로 감지되며 즉시 조치를 취하거나 드리프트를 더 자세히 살펴볼 수 있습니다. 자세한 내용은 Mix Modeler 설명서에서 [모델 드리프트](https://experienceleague.adobe.com/ko/docs/mix-modeler/using/models/insights#model-drift){target="_blank"}를 참조하십시오.
